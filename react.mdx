## Add Shippr to your React project

<CodeGroup>
```js YARN
yarn add @shippr/react
```
```js NPM
npm install @shippr/react
```
</CodeGroup>




```jsx simple presence component
import React from "react"
import { ChannelPresence } from "@shippr/react"

export default function Demo() {
  return (
    <div>
      <ChannelPresence
        channelId="my-channel"
        apiKey="APIKEY"
        appId="APPID"
        mode="stack"
        stackLimit={5}
      />
    </div>
  )
}

          ```
